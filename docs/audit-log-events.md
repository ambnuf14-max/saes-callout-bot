# Audit Log — события

## Что сейчас логируется

### 🚨 Каллауты
| Событие | Поля |
|---|---|
| **Каллаут создан** | ID, подразделение, фракция, канал, создатель, кратко об инциденте, статус VK/TG уведомлений |
| **Каллаут закрыт** | ID, подразделение, закрыл, канал, длительность, причина |

### 📣 Реагирование
| Событие | Поля |
|---|---|
| **Ответ из VK** | ID каллаута, подразделение, имя и ID пользователя VK |
| **Ответ из Telegram** | ID каллаута, подразделение, имя и ID пользователя Telegram |
| **Реагирование из Discord** | ID каллаута, подразделение, mention пользователя |

### 🏛️ Фракции и подразделения
| Событие | Поля |
|---|---|
| **Фракция добавлена** | Название, роль, VK беседа |
| **Фракция обновлена** | Название, список изменений |
| **Фракция удалена** | Название |
| **Подразделение добавлено** | Название, фракция |
| **Подразделение обновлено** | Название, фракция |
| **Подразделение удалено** | Название, фракция |

### 📋 Типы фракций и шаблоны
| Событие | Поля |
|---|---|
| **Тип фракции создан** | Название, описание |
| **Тип фракции обновлён** | Название, список изменений |
| **Тип фракции удалён** | Название |
| **Шаблон подразделения добавлен** | Тип фракции, название шаблона |

### ✅ Система одобрения изменений (pending)
| Событие | Поля |
|---|---|
| **Запрос на создание подразделения** | ID, тип, фракция, кто запросил, детали diff |
| **Запрос на обновление подразделения** | ID, тип, фракция, кто запросил, детали diff |
| **Запрос на удаление подразделения** | ID, тип, фракция, кто запросил, детали diff |

| **Изменение отклонено** | Тип, фракция, кто отклонил, причина, детали |
| **Изменение отменено** | Тип, фракция, детали |

### ⚙️ Настройки сервера
| Событие | Поля |
|---|---|
| **Настройки сервера обновлены** | Список изменений |
| **Лидерская роль добавлена** | Роль |
| **Лидерская роль удалена** | Роль |
| **Роль каллаутов добавлена** | Роль |
| **Роль каллаутов удалена** | Роль |

### 🔗 Интеграции
| Событие | Поля |
|---|---|
| **VK беседа привязана** | Фракция, подразделение, VK Chat ID, название беседы |
| **Telegram группа привязана** | Фракция, подразделение, Telegram Chat ID, название группы |

## Что можно добавить

### 🚨 Каллауты
- **Каллаут автоматически закрыт по таймауту** — отдельное событие от ручного закрытия, чтобы видеть сколько инцидентов висело без ответа
- **Повторное уведомление отправлено** — если была функция напоминания подразделению

### 📣 Реагирование
- **Подразделение отказалось реагировать** — если добавить кнопку отказа

### 👤 Верификация и участники
- **Пользователь верифицирован через VK** — кто верифицировался, какой аккаунт VK привязан
- **Пользователь верифицирован через Telegram** — аналогично
- **Верификационный токен создан** — кем, для кого
- **Токен просрочен/использован**

### 🏛️ Фракции и подразделения
- **Подразделение поставлено на паузу** (`is_accepting_callouts = false`) — кто и когда отключил приём каллаутов
- **Подразделение снято с паузы** — кто и когда включил обратно
- **Presence Asset установлен** — кто настроил, для какого подразделения, какое имя ассета

### ⚙️ Настройки сервера
- **Канал инцидентов изменён** — старый и новый канал
- **Категория каналов изменена** — старая и новая
- **Задержка удаления канала изменена** — старое и новое значение (в минутах)

### 🔗 Интеграции
- **VK беседа отвязана** — фракция, подразделение
- **Telegram группа отвязана** — фракция, подразделение
- **Ошибка отправки уведомления VK** — каллаут, подразделение, текст ошибки (сейчас только в логах файла)
- **Ошибка отправки уведомления Telegram** — аналогично

### 🔐 Безопасность / доступ
- **Попытка несанкционированного доступа** — кто пытался нажать кнопку без роли (respond, close)
- **Использование /history** — кто и когда смотрел историю (опционально, если нужен trail)
